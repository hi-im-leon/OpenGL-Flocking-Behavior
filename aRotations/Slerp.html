Interpolating rotations using splines!

In this assignment segment, you will implement
  • linear and cubic splines for quaternions
This framework contains two demos but this week, we will focus on only one of them: the Curve Demo. The Curve Demo uses a rotational spline to animate the orientation of our yellow teapot. In this assignment, you will implement linear spherical interpolation using Slerp and cubic spherical interpolation using Squad. Below is a screenshot.

The demo controls panel lets you select the current demo and set properties for either demo.

Assignment

(50 points) Quaternion splines. The basecode includes a class ASplineQuaternion whose interface mirrors the spline you worked with for vectors in the previous assignment. In this assignment, you will complete the implementation by implementing Slerp and Squad. See RotationViewer::drawDemo2() to debug your routines.
  1. (20 points) Linear interpolation.  Implement Quaternion::Slerp()
  2. (30 points) Cubic interpolation.
    1. Implement Quaternion::Squad()
    2. Implement Quaternion::Intermediate()
    3. Compare the animation of the teapot using linear versus cubic interpolation. What are the differences? (Note: The keys used for the demo are hard-coded in the RotationViewer constructor.)
